# Windows 95 Nostalgia Personal Website

![Windows 95 Logo](https://i.imgur.com/sqf7tWx.jpeg)

*"Where do you want to go today?"*

A personal website built to look and feel like Windows 95.

## Why Windows 95?

Windows 95 came out before my time, but it has always fascinated me. It was the moment personal computing went mainstream. The Start menu, the taskbar, the beveled buttons: design decisions made under real constraints that somehow became iconic. As someone who builds software for a living, I have a lot of respect for that kind of craft.

This site is my tribute to it.

## Stack

Plain HTML, CSS, and JavaScript. No React, no Tailwind, no bundler. Deliberately so.
